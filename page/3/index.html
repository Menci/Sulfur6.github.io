这个游戏的地图可以看作一棵包含 n n n 个结点和 n−1 n - 1 n−1 条边的树，每条边连接两个结点，且任意两个结点存在一条路径互相可达。树上结点编号为从 1 1 1 到 n n n 的连续正整数。
