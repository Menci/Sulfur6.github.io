题目大意

给定一棵 n n n 个节点的树，每个节点上有颜色，要求支持两种操作：

  1. 询问两点 (u,v) (u, v) (u,v) 之间有多少段颜色
  2. 将两点 (u,v) (u, v) (u,v) 之间所有的节点的颜色染成 c c c

题解

树链剖分线段树维护，注意线段树每个节点要维护这个区间最左边和最右边位置上节点的颜色，两段区间的答案合并时如果左区间的最右节点和右区间的最左节点颜色相同的话答案要减 1 1 1。

树剖统计答案的时候也要注意这一点。
